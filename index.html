Malware Analysis Setup: REMnux and FlareVM

In my free-time I like dissecting malware, to be able to do that in a safe environment. I set up two virtual machines, one is set up with REMnux and the other configured with FlareVM. REMnux is a Linux-based toolkit for malware reverse-engineering and analysis, and FlareVM is a Windows environment for forensic analysis and malware research. With the combination of these two tools, I can analyse virus samples and investigate the impact and behaviour of the viruses.
