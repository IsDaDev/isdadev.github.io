SigSafe AV (Work in Progess)

SigSafe SBMD

SigSafe SBMD utilizes signature-based detection to help secure your machine by detecting malware. It pulls signature information from a database and compares the signature info to files on your machine. While the database may not contain the latest signatures, it can still be effective in identifying known malware by checking if files on your machine match the known signature associated with the harmful software.

SigSafe WebGuard: Safe Browsing Made Easy

SigSafe WebGuard is intended to provide a safe browsing experience. Before connecting to a web page, WebGuard checks the site against its list of URLs that it considers known malicious URLs. If it matches one on the list, WebGuard will prevent you from connecting to the site to protect you. While the flagged URL list may not always be current, WebGuard acts as an effective layer of protection for your browsing experience.

Malware Analysis Setup: REMnux and FlareVM

In my free-time I like dissecting malware, to be able to do that in a safe environment, I set up two virtual machines, one is set up with REMnux and the other configured with FlareVM. REMnux is a Linux-based toolkit for malware reverse-engineering and analysis, and FlareVM is a Windows environment for forensic analysis and malware research. With the combination of these two tools, I can analyse malware samples and investigate the impact and behaviour of the malware.
